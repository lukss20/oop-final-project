Project: Quiz Web Application

This project is a Java-based web application developed as part of a university team project. The system allows users to create, take, and evaluate quizzes, providing a complete platform for interactive learning and assessment.

Features
User registration and login system
Create, edit, and delete quizzes
Take quizzes with multiple question types
Automatic scoring and result calculation
View past results and user activity
Filter and browse quizzes
Technologies Used
Backend: Java Servlets, JDBC
Frontend: JSP, HTML/CSS
Database: MySQL
Server: Apache Tomcat

Architecture
The application follows a layered structure:
Servlets handle request/response logic
DAO (Data Access Objects) manage database operations
JSP pages render dynamic content
MySQL database stores users, quizzes, and results

Purpose
The goal of this project was to build a full-stack web application, combining backend development, database design, and frontend integration while working collaboratively in a team environment.
